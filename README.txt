DESKRIPSI
---------
Website resmi pemerintahan Nagari Silongo yang modern, profesional,
dan responsif. Dibangun menggunakan HTML murni, CSS, JavaScript,
dan Tailwind CSS CDN - tanpa framework apapun.

CARA MENGGUNAKAN
----------------
1. Buka file "index.html" langsung di browser
2. Website akan langsung tampil dengan semua fitur aktif
3. Tidak perlu instalasi apapun!

STRUKTUR FILE
-------------
nagari-silongo-website/
├── index.html          ← File utama website
├── style.css           ← Semua custom CSS & animasi
├── script.js           ← Semua JavaScript (popup, carousel, dll)
├── assets/
│   ├── images/         ← Letakkan foto/gambar di sini
│   ├── icons/          ← Letakkan ikon di sini
│   └── backgrounds/    ← Letakkan background images di sini
└── README.txt          ← File ini

FITUR WEBSITE
-------------
✅ Navbar fixed - transparan di atas, solid saat scroll
✅ Mobile hamburger menu
✅ Hero section fullscreen dengan CSS art
✅ Statistik desa dengan counter animasi
✅ Profil nagari dengan popup 3 Jorong
✅ Pemerintahan - 4 pejabat dengan popup detail
✅ Potensi desa - 5 bidang dengan popup detail
✅ Wisata & Budaya - carousel interaktif + lightbox
✅ Berita terkini - 4 artikel dengan popup lengkap
✅ Galeri foto - masonry grid dengan lightbox navigasi
✅ Kontak - info + Google Maps embed + WhatsApp
✅ Footer profesional 3 kolom
✅ Floating WhatsApp button
✅ Scroll to top button
✅ Smooth scrolling
✅ Scroll reveal animations
✅ Responsive mobile/tablet/desktop
✅ Loading animation

TEKNOLOGI
---------
- HTML5
- CSS3 (Custom Properties, Grid, Flexbox, Animations)
- JavaScript (Vanilla ES6, IntersectionObserver)
- Tailwind CSS CDN (https://cdn.tailwindcss.com)
- Google Fonts - Poppins

CARA DEPLOY KE VERCEL
---------------------
1. Buat akun di vercel.com
2. Klik "Add New Project"
3. Upload folder ini atau push ke GitHub dulu
4. Vercel akan otomatis deploy - tidak perlu konfigurasi!

CARA UPLOAD KE GITHUB
---------------------
1. Buat repository baru di github.com
2. Upload semua file dari folder ini
3. Pastikan index.html ada di root repository
4. Enable GitHub Pages di Settings > Pages (pilih branch main)
